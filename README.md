 Flipping Card Project Details:

A simple and interactive **Flipping Card UI** built using HTML & CSS.  
This project demonstrates how to create a card that flips on hover to reveal additional content.



 🚀 Features
- Flip animation using CSS
- Clean and responsive design
- Smooth hover effect
- Beginner-friendly project


 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox & Transforms)



💡 How It Works
The card flips using CSS `transform` and `transition` properties.  
When the user hovers over the card, it rotates to show the back side.
